   - Paths like `/` or empty become group `"root"`

2. Add `_relative_time(ts_str)` function and register as Jinja2 filter:
   - If `ts_str` is None or empty, return `"\u2014"` (em dash)
   - Parse with `datetime.strptime(ts_str, "%Y-%m-%d %H:%M:%S").replace(tzinfo=timezone.utc)`
   - Compute delta from `datetime.now(timezone.utc)`
   - Format as: `<60s` -> "{N}s ago", `<3600s` -> "{N}m ago", `<86400s` -> "{N}h ago", else -> "{N}d ago"
   - Register: `_env.filters["relative_time"] = _relative_time`

**home.html** -- Complete rewrite. The template receives context vars: `groups` (dict of prefix -> list[dict]), `debug_route`, `method_filter`, `sort`, `total_routes`.

Structure:
1. Extends `base.html`
2. Topbar with `FlowSurgeon` logo (no breadcrumb on home -- it IS the root)
3. Method filter pills bar wrapped in Alpine.js `x-data="{ methodFilter: 'ALL' }"`:
   - Six pills: ALL, GET, POST, PUT, DELETE, PATCH
   - Each pill: `<button class="pill" :class="methodFilter === 'X' ? 'pill-active' : ''" @click="methodFilter = 'X'">X</button>`
4. Route count summary: `{{ total_routes }} routes`
5. For each group in `groups.items()`:
   - Group header: `<div class="route-group-hd">{{ prefix }}</div>` (styled as section divider)
   - For each route in group:
     - Apply `x-show="methodFilter === 'ALL' || methodFilter === '{{ route.method }}'"` on INDIVIDUAL route rows (not group level -- per Pitfall 6)
     - Route row as `<a>` tag linking to `{{ debug_route }}?method={{ route.method }}&path={{ route.path | urlencode }}`
     - Row class: `class="tr {% if not route.has_data %}tr-no-traffic{% endif %}"`
     - Left side: method badge `<span class="method {{ route.method | method_class }} {% if not route.has_data %}method-muted{% endif %}">{{ route.method }}</span>` + path in monospace
     - Right side (only if `route.has_data`):
       - `{{ route.count }} reqs`
       - `{{ "%.0f"|format(route.latest_duration_ms) }}ms` with duration_class
       - `{{ route.latest_query_count }}q`
       - Status badge: `<span class="status {{ route.latest_status | status_class }}">{{ route.latest_status }}</span>`
       - `{{ route.last_request_time | relative_time }}`
     - If not `route.has_data`: show `No requests` in dim text, right-aligned
6. Empty state: if `total_routes == 0`, show "No routes discovered" message
7. Group headers should also respect Alpine filter: show group if ANY child row would be visible. Use `x-show` with a computed check or just always show headers (simpler -- empty groups with all rows hidden will collapse visually).
